## Moving Control Room Moment

Aslam was outside, traveling for curtain fixing work.

Curtain fixing work may take only one hour, but travel takes four to five hours.

During travel, while the driver was driving, Aslam was sitting behind in the car using mobile Termux like a world control room.

The goal was clear:

Control Aslam-owned devices safely from anywhere, even when IP address changes because of:

- room WiFi
- office WiFi
- mobile data
- VPN
- router changes
- travel
- Pi5 being on or off

## Core Idea

IP is only a door number.

A changing IP is not a reliable key.

The real master key must be identity-based.

## NOVA LINK Master-Key Idea

NOVA LINK will control only Aslam-owned devices using:

- Aslam mobile private key
- trusted device identity
- signed command packets
- device-side verification
- allowlisted commands
- approval for risky actions
- full logs
- backup before changes
- rollback after failure

## Safe Rule

This system is not for hacking other people or unknown devices.

It is for Aslam's own devices only.

The owner must be verified by key, not by IP.
